# bOS
bOS is a new windows cmd client! Here is everything you need to know
# Commands
Commands:
print : prints text;
(Example : print bOS
bOS)
mkdir : makes new folder;
mkfile : makes new file;
cd : opens folder;
update : updates bOS;
reinstall : reinstalls bOS;
ip : shows info by IP address;
set: settings
(startsfx: start music(sfx1,sfx2,sfx3)
menumusic: menu music(music1,music2,music3));
writetofile: writes to file;
replace: replaces file or data in file;
deldir: deletes folder;
delfile: deletes file;
rename: renames files and folders;
location: shows your location by ip;
time: shows current system time;
utc: shows current utc time;
notepad: opens boba notepad;
ls: shows all files in current directory (blue:files, yellow:folders, red:system folders, green:system files);
pwd: shows current directory;
python: executes python scripts
install: installs apps
clear: clears your terminal
set:
	rootpassw: changes root password
	uwarn: changes show update warning tue/false
	stcom: changes able to start startup caommands true/false
	stroot: changes root startup true/false
root: gives root if you know password
	-root: cancels your root
	-check: checks root
admin: gives admin rights to bOS
beditor: opens beditor
color: changes console color (may not work in debug mode)
cmatrix: matrix digital rain (q: quit; r: restart; c: change color mode)
	green: green rain with white ending (standart)
	classic: white rain
	rgb: blue color added
hacking: hacking pentagon. Jk, but it make your screen goes crazy
neofatch: shows system info
run: runs command in your kernel
